# Contributor Covenant Code of Conduct

## Our Pledge
We pledge to make participation in this project respectful and harassment-free.

## Standards
Examples of positive behavior:
- Respectful, technical feedback
- Constructive issue reporting
- Professional collaboration

Examples of unacceptable behavior:
- Harassment, insults, or personal attacks
- Publishing private information
- Deliberate disruption of project workflows

## Enforcement
Project maintainers are responsible for clarifying and enforcing standards.
Contact: conduct@ctrl.example.com
